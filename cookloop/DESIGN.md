# COOKLOOP Design System

COOKLOOP is a recipe sharing app where users can discover, create, save and share recipes.

## Pages
- Home
- Search
- Create
- Inbox
- Profile

## Components
- Navbar
- Footer / Bottom Navigation
- Recipe Card
- Search Box
- Filter Chips
- Message Row
- Profile Grid
